# 快速开始

> 安装完成后，按以下步骤配置即可开始使用。全程不需要任何技术知识。

## 第一步：填写本店信息

点击右上角控制后台进入管理后台：

1. 输入你的**酒店名称**（全称），点击保存
2. 点击**自动定位**，系统会根据酒店名自动获取位置
3. 确认地图上的位置是否正确

> 这一步很重要，Agent 需要知道你是哪家酒店，才能帮你查数据、分析竞品。

## 第二步：设置时区

右上角有时区选择，默认是北京时间。如果你的酒店不在中国大陆，切换到当地时区即可。

## 第三步：登录 OTA 平台

在左侧浏览器中，登录你常用的商家后台：

| 平台 | 地址 |
|------|------|
| 携程后台 | ebooking.ctrip.com |
| trip后台 | ebooking.trip.com |
| 美团后台 | me.meituan.com |
| Booking后台 | admin.booking.com |

**至少登录一个平台。** 登录越多，Agent 能帮你做的事越多。

登录后保持页面打开，不要关掉。

## 第四步：添加竞品（可选但推荐）

管理后台 → 竞品管理：

1. 第一步的定位完成后，点击**搜索周边酒店**
2. 从搜索结果中选择你的竞品，点击添加
3. 也可以手动输入竞品名称添加

添加后 Agent 就能帮你对比竞品价格了。

## 配置完成，可以开始用了

直接在右侧对话框跟 Agent 说话就行，比如：

- "今天房价多少？"
- "这周入住率怎么样？"
- "帮我跟竞品比一下价格"
- "帮我调个价"
- "帮我写评价回复"（需要先在左侧打开评价页面）

## 各功能需要什么

| 想做什么 | 需要先完成 |
|---------|-----------|
| 查房价、入住率 | 登录对应平台 |
| 竞品价格对比 | 添加竞品 + 登录平台 |
| 智能调价 | 登录携程后台 |
| 写评价回复 | 在左侧浏览器打开评价页面 |
| 自动定时采集 | 管理后台开启对应平台的定时任务 |

## 常见问题

**Agent 说"数据获取失败"？**
→ 最常见的原因是左侧浏览器里没登录或登录过期了，重新登录一下就好。

**Agent 说"竞品数据暂无"？**
→ 先在管理后台添加竞品，然后确保登录了对应的公网平台（如 hotels.ctrip.com）。

**想让系统自动采集数据？**
→ 管理后台 → 数据渠道配置 → 开启对应平台的定时任务，系统会按时自动采集。
