# 登录页入口屏蔽说明

## 当前状态

已屏蔽：**注册入口** 和 **Google 登录入口**（2026-05-09）

用户只能通过我们提供的账号密码进行登录。

## 屏蔽位置

文件：`src/renderer/components/login-page.ts`，`createLoginPageDOM()` 方法内。

### 1. 注册入口（"没有账号？注册"链接）

```html
<div class="form-link" style="display:none">
  ${lt('noAccount')}<a href="#" id="showRegister">${lt('register')}</a>
</div>
```

### 2. Google 登录（分割线 + 按钮）

```html
<div class="google-login-divider" style="display:none">
  <span>${lt('or')}</span>
</div>
<button class="google-login-btn" id="googleLoginBtn" style="display:none">
  ...
</button>
```

## 如何恢复

在上述文件中搜索 `style="display:none"`，删除以下三处的 `style="display:none"`：

1. `<div class="form-link" style="display:none">` → `<div class="form-link">`
2. `<div class="google-login-divider" style="display:none">` → `<div class="google-login-divider">`
3. `<button class="google-login-btn" id="googleLoginBtn" style="display:none">` → `<button class="google-login-btn" id="googleLoginBtn">`

修改后重新编译即可恢复注册和 Google 登录功能。
