# 测试条例

---

## 一、安装与登录

- [ ] 首次启动正常，显示登录页面
- [ ] 能正常注册账号并进入主界面
- [ ] 能正常登出、再登录
- [ ] 记住密码后重新打开应用，可一键登录
- [ ] Google 登录正常（弹出 Chrome 完成授权后自动登录）

---

## 二、浏览器

- [ ] 地址栏输入网址能正常访问
- [ ] 前进、后退、刷新正常
- [ ] 新建/关闭/切换标签页正常
- [ ] 能在浏览器中登录携程 ebooking、美团商家等 OTA 后台
- [ ] 登录后切换标签再切回来，登录状态不丢失

---

## 三、酒店信息配置

- [ ] 控制后台正常打开
- [ ] 填写酒店名称并保存，显示"已配置"
- [ ] 自动定位成功，显示酒店地址
- [ ] 能添加多个标准房型，显示为蓝色标签
- [ ] 能删除已添加的房型

---

## 四、竞品配置

> 前提：酒店名称和定位已完成

- [ ] 搜索周边酒店能正常返回结果
- [ ] 切换搜索半径后结果范围有变化
- [ ] 能勾选并添加竞品，竞品管理列表正常显示
- [ ] 已添加的竞品不能重复添加
- [ ] 能删除已添加的竞品

---

## 五、数据采集

> 前提：已在浏览器中登录对应 OTA 平台

- [ ] 数据渠道页面正常显示各平台和渠道
- [ ] 能开启/关闭渠道，能配置采集时间并保存
- [ ] 采集后数据面板中能看到对应数据

---

## 六、数据展示面板

> 前提：至少完成过一次数据采集

- [ ] 价格日历正常展示，房型×日期有价格数据
- [ ] 鼠标悬停能看到底价、含早等详情
- [ ] 切换日期范围（今天/7天/30天）表格正常变化
- [ ] 按平台筛选正常
- [ ] 房态日历正常展示，颜色区分空房率

---

## 七、AI 对话

- [ ] 发消息能正常收到 AI 回复
- [ ] 多轮对话保持上下文连贯
- [ ] 能新建对话、切换历史会话
- [ ] AI 执行任务时显示进度信息

---

## 八、界面偏好

- [ ] 中英文切换正常
- [ ] 深色/浅色主题切换正常
- [ ] 拖拽分隔条能调节左右面板宽度
- [ ] 关闭应用再打开，之前的配置（酒店信息、渠道设置、竞品）不丢失

---

## 九、开放性测试

请按日常习惯自由使用，记录遇到的任何问题：

- 模拟日常工作：查看房价 → 对比竞品 → 问 AI 调价建议
- 在浏览器打开 OTA 评价页，让 AI 帮忙生成回复
- 同时开启多个平台采集，观察是否正常
- 随意操作（快速点击、开多标签、输入长消息等），观察是否卡顿或报错

---

## 体验反馈

1. 安装过程顺利吗？
2. 各功能好找吗？有没有不知道怎么用的地方？
3. AI 回答速度和质量如何？
4. 数据面板清楚吗？想看的数据都有吗？
5. 有什么功能希望增加或改进？
