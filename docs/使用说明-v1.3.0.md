# 使用说明

## 一、安装

1. 双击 **Hotel AI Browser Setup 1.3.0.exe**，按向导提示完成安装
2. 可自行选择安装位置
3. 安装完成后桌面和开始菜单会出现快捷方式

> 国内网络环境需开梯子

---

## 二、注册与登录

首次打开应用会进入登录页。

### 注册账号

1. 点击页面下方 **注册**
2. 填写用户名、密码、确认密码、显示名称
3. 点击注册，注册成功后自动进入主界面

### 登录

- 输入用户名和密码，点击 **登录**
- 可勾选 **记住密码**，下次打开时在页面顶部直接点击账号即可一键登录
- 也可点击 **Google 图标** 使用 Google 账号登录（会弹出 Chrome 窗口完成授权）

---

## 三、界面介绍

登录成功后进入主界面，左右分屏布局：

- **左侧 — 浏览器**：与普通浏览器一样，可以访问任意网页、登录各 OTA 平台后台
- **右侧 — AI 助手**：与 AI 对话，可以查数据、问问题、让 AI 协助操作
- **中间分隔线**：可以左右拖拽调节两侧宽度

顶部功能栏从左到右：
- 标签栏（多标签浏览）、地址栏、收藏按钮
- **中/EN**：切换中英文界面
- **月亮/太阳图标**：切换深色/浅色主题
- **控制后台**：打开管理面板（配置酒店信息、查看数据等）
- 用户名 + 登出按钮

---

## 四、首次使用 — 必要配置

安装后**第一件事**是完成以下配置，否则功能无法正常使用。

### 第 1 步：配置本店信息

1. 点击顶部 **控制后台** 按钮
2. 左侧菜单选择 **酒店信息配置**
3. 在"本店信息"区域输入您的酒店名称，点击 **保存**
4. 点击 **自动定位**，系统会自动定位当前酒店地址
5. 定位成功后会显示酒店地址（绿色图标）

> 酒店名称和定位是后续竞品搜索、数据关联的基础，请确保准确。

### 第 2 步：配置标准房型

在同一页面的"标准房型管理"区域：

1. 输入您酒店的房型名称（如"豪华大床房"、"标准双床房"），按回车或点击添加
2. 逐个添加所有房型
3. 添加的房型会以蓝色标签展示，点击 × 可删除

> 标准房型用于将不同 OTA 平台上的房型名称匹配到统一名称，确保数据面板中的价格对比准确。

### 第 3 步：添加竞品酒店（可选）

定位成功后，页面下方会出现"竞品发现"区域：

1. 选择搜索半径（1km / 2km / 3km / 5km）
2. 点击 **搜索周边酒店**
3. 在搜索结果中勾选要关注的竞品酒店
4. 点击 **添加选中为竞品**
5. 已添加的竞品会出现在"竞品管理"列表中

系统会自动解析竞品在各 OTA 平台上的 ID（显示 ✅ 表示解析完成）。开启公网价格采集后，竞品的房价也会一并采集。

### 第 4 步：登录 OTA 平台

**这是数据采集的前提条件。** 在左侧浏览器中登录您使用的 OTA 平台后台：

| 平台 | 说明 |
|------|------|
| 携程 ebooking | 用于采集携程后台价格 |
| 美团商家后台 | 用于采集美团后台价格 |
| PMS 系统 | 用于采集当前/未来房态信息 |
| 携程公网 | 用于采集携程公网展示价 |
| Trip.com | 用于采集携程国际渠道价格 |
| Booking | 用于采集 Booking 公网价 |

- 登录一次后状态会自动保持，过期后需重新登录
- 数据采集的前提需要保证登录状态

### 第 5 步：开启数据采集

1. 打开 **控制后台** → 左侧菜单栏选择 **数据渠道配置**
2. 找到要开启的平台和渠道，打开开关
3. 点击渠道旁的展开箭头，设置采集时间：
   - **间隔模式**：每隔 N 分钟/小时自动采集一次
   - **固定时间模式**：设定每天的具体执行时刻（可添加多个时间点）
4. 点击 **保存调度**

> 最多可同时开启 4 个平台。开启后系统会按设定时间自动执行数据采集。
>
> 第一次采集需初始化，会比较慢，初始化完成后后续采集正常使用

## 五、日常使用

### 5.1 查看数据面板

打开 **控制后台** → **数据展示面板**：

- **价格日历**：表格形式展示各房型各日期的价格
  - 可按平台筛选（全部 / 携程 / 美团 / Booking）
  - 鼠标悬停可看到底价、含早信息、在售状态等详情
  - 本店和竞品分区显示，方便对比
- **房态日历**：显示各房型各日期的剩余房间数
  - 绿色 = 充足，黄色 = 紧张，红色 = 很少，黑色 = 无房
- **PMS 指标**：入住率、ADR（平均房价）、RevPAR（每可用房收入）
- **价格趋势图**：折线图展示价格随时间的变化
- 顶部可切换日期范围：今天 / 未来 7 天 / 14 天 / 30 天

### 5.2 与 AI 助手对话

在右侧对话面板中直接输入问题，例如：

- "最近三天的房价情况如何？"
- "和竞品对比，我们的价格是偏高还是偏低？"
- "帮我看看携程上这个评论怎么回复比较好"
- "下周末的房间预订情况怎样？"

按 **Enter** 发送，**Shift+Enter** 换行。

AI 执行采集任务时，面板上方会显示实时进度和截图。

点击右上角气泡图标可以新建一个对话。

### 5.3 数据采集的运行状态

当定时任务执行时：
- 标签栏会出现带 🤖 图标的标签，表示采集正在后台运行
- 数据渠道页面对应的渠道会显示蓝色"获取中"动画
- 采集完成后数据面板会自动更新

### 5.4 管理对话记录

控制后台 → **会话管理**：
- 查看所有历史对话
- 点击可切换到对应对话
- 不需要的对话可以删除

## 六、常见问题

### 采集任务没有执行？
- 检查渠道开关是否已开启
- 后台渠道（携程 ebooking、美团商家）需要先在浏览器中登录
- 如果登录状态过期，需要重新在浏览器中登录一次

### 数据面板没有数据？
- 需要至少完成一次数据采集后才有数据
- 检查是否已开启渠道并等待执行

### AI 不回复 / 回复很慢？
- 检查网络连接是否正常
- AI 处理复杂问题时可能需要稍等

### 竞品搜索没有结果？
- 确认酒店名称和定位是否正确
- 尝试增大搜索半径

### 启动时提示"版本不符"？
- 需要更新到最新版本，请联系管理员获取新安装包

---

## 七、注意事项

1. **保持应用运行**：定时采集任务只在应用开启时执行，关闭应用后任务暂停
2. **定期检查登录状态**：后台渠道的登录会过期，如果发现采集失败，请重新在浏览器中登录对应平台
